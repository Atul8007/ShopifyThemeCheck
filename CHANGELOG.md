All notable changes to this project will be documented in this file.

## [1.0.0] - 2023-10-27

### Added
- Initial release of Shopify Theme Check extension.
- Feature: Detect `schema_name` from Shopify theme scripts.
- Feature: Fallback detection for theme ID and developer name.
- Feature: Direct links to Shopify Theme Store and ThemeForest.
- UI: Minimal popup interface.
